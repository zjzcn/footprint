安装步骤指导

一、手动安装

1)解压缩安装文件包到你的应用服务器的根目录(推荐)或指定目录

2)备份xxx目录下xxx.properties文件

3)在应用服务器的启动脚本中增加：

改！！-javaagent:/path/to/xxx/xxx-agent-java.jar

注：“/path/to/”是示例路径，请用户根据自身不同的环境修改正确的目录。

二、启动或者重新启动应用服务器

1)应用服务器重新启动，云迹server探针监控才能生效。

2)5分钟后，当您的java应用服务有http请求进入，性能数据将发送到云迹Server
